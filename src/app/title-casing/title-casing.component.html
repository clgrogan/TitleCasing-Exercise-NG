Title Casing | TitleCasePipe

The application takes the user input and converts it to title case*.

Uses the custom pipe TitleCasePipe (keyword: titlecase).

| titlecase: minorWords {{ userInput | lowercase | titlecase: minorWords }}
| titlecase {{ userInput | lowercase | titlecase }}

*A string is considered to be in title case if each word in the string is either capitalized with only the first letter in uppercase. Except in the case of a minor word, which will be all lower case unless it is the first word of the string, in which case it will be capitalized like a 'normal' word.

Minor words from title-casing component: {{ minorWord }}
